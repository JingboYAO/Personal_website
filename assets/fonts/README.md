# Fonts

Place WOFF2 font files here.
Required fonts based on `style-dictionary.json`:

- Lato-Regular.woff2
- Lato-Bold.woff2
- Merriweather-Regular.woff2
- Merriweather-Bold.woff2

Ensure each file is < 50kB.
Update `assets/css/style.css` or create `assets/css/fonts.css` to reference them via `@font-face`.
